# Products Sales Data Analysis Dashboard

## Task 2 - Medium

This project was completed as part of the **Hunar Intern Data Analytics Internship**. The dashboard provides insights into product sales, profit, and business performance using interactive Power BI visualizations.

## Tools Used
- Power BI
- Microsoft Excel / CSV
- Power Query
- DAX
## Dashboard Features
- Total Sales KPI
- Average Sales Per Day
- Total Units Sold
- Total Profit
- Best Selling Product Analysis
- Sales by Product
- Profit by Country
- Daily Sales Trend
- Interactive Filters (Date, Country, Product, Segment)
