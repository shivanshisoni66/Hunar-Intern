# IPL Score Analysis Dashboard (2008–2019)

## Internship Project

This project was completed as part of the Hunar Intern Data Analytics Internship.

## Tools Used
- Power BI
- Excel
- CSV

## Features
- KPI Cards
- Season Wise Matches
- Team Wins Analysis
- Toss Winner Analysis
- Top Wicket Takers
- Top Run Scorers
- Winning Margin Analysis
- Interactive Filters
